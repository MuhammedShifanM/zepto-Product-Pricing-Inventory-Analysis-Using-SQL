# zepto-Product-Pricing-Inventory-Analysis-Using-SQL

# Project Overview
This project focuses on analyzing the retail dataset of Zepto using PostgreSQL to generate meaningful business insights related to product pricing, discounts, stock availability, and inventory performance. The analysis helps understand product trends, category-wise distribution, and discount strategies used in the quick commerce industry.

# Objectives
Analyze product pricing and discount patterns
Identify high and low inventory products
Evaluate category-wise product distribution
Detect out-of-stock products and inventory gaps
Perform data cleaning and transformation using SQL
Generate business insights through analytical SQL queries
